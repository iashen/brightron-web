Maternal Health Care System

Maternal Health Care System

The Maternal Health Care System is a web based system and it will facilitate the functions followed by Public Health Mid Wives, by maintaining the details of pregnant mothers and managing regular visits to houses. Registration of pregnant mothers and eligible families is done by using an Android application with their GPS locations. The system consist of an Automated Notification system and SMS reminders to inform about the clinic dates and upcoming deliveries. Maternal Health Care System enables Cloud Storage and File Sharing among users. Report Generation functionalities can be used to analyze data. Due to the uses of Mobile Responsive Interfaces these functions can be monitored by any Mobile Device and from any location.
